Overview
This project showcases an interactive Tableau dashboard analysing Airbnb listing data to generate insights into pricing, revenue trends, and geographical distribution. It demonstrates an end-to-end approach to data analysis, from data preparation in Excel to visualisation and insight generation.

Key Features
* Price by Zipcode: Comparative analysis of average listing prices across locations
* Revenue Trend: Time-series analysis highlighting seasonal patterns and performance over the year
* Geographical View: Map-based visualisation of pricing distribution by area
* Average Price per Bedroom: Insight into how pricing scales with property size
* Listings Breakdown: Distribution of listings by number of bedrooms

Tools & Technologies
* Tableau (Data visualisation and dashboard development)
* Microsoft Excel (Data preparation and source data)

Key Insights
The analysis highlights variations in pricing across different locations, identifies higher-value areas, and shows how property size influences average price. It also provides an overview of revenue trends over time, supporting data-driven decision-making.

Repository Contents
* Tableau workbook (.twb)
* Source dataset (Excel) from Kaggle

Purpose
This project was developed to strengthen my data visualisation and analytical skills, with a focus on presenting clear, actionable insights. It reflects my interest in applying data analysis techniques to real-world scenarios and building user-friendly dashboards for decision support.
